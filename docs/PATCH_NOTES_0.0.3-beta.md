# Mabinogi Overlay 0.0.3-beta Patch Notes

기준 버전: `0.0.2-beta`

## 핵심 변경점

- 프로필 저장 방식을 수동 저장에서 변경 사항 자동 저장 방식으로 전환했습니다.
- 프로필 선택, 생성, 불러오기 기능을 메인 화면의 전용 프로필 영역으로 이동했습니다.
- 오버레이 시작과 중지 기능을 별도의 오버레이 조작 영역으로 분리했습니다.
- 앱 세션 로그가 portable 실행 파일 옆의 `Logs` 폴더에 저장되도록 변경했습니다.

## 프로필 및 레이아웃

- 프로필의 `저장` 버튼을 `생성` 버튼으로 변경했습니다.
- 프로필을 생성한 뒤 후보 슬롯, 섹션, 오버레이 배치 및 레이아웃 설정 변경 사항이 자동으로 저장됩니다.
- Layout Editor에서 Apply를 실행하거나 편집 창을 닫을 때 변경 사항이 저장됩니다.
- 앱 종료 시 실제 변경 사항이 있을 때만 저장하여 빈 초기 상태가 기존 프로필을 덮어쓰지 않도록 했습니다.
- 다른 프로필을 선택하는 것만으로 기존 프로필이 덮어써지지 않도록 활성 프로필과 불러오기 대상을 구분했습니다.

## UI 및 사용성

- 상단 명령 영역과 본문 사이에 프로필 및 오버레이 조작 영역을 추가했습니다.
- 새 영역에 맞춰 기본 창 높이와 최소 높이를 조정했습니다.
- 메인 화면의 주요 영역 간 여백을 줄여 화면 사용 효율을 높였습니다.
- Settings의 프로필 저장 경로에서 Browse 및 Default 버튼이 잘리던 문제를 수정했습니다.
- 슬롯 목록 및 오버레이 레이아웃 Clear 실행 전에 확인창을 표시합니다.
- 빈 목록이나 빈 레이아웃을 초기화할 때 표시되는 상태 문구의 한국어 번역을 보완했습니다.

## 수정 사항

- 슬롯 목록에서 후보를 삭제해도 연결된 오버레이 슬롯이 남던 문제를 수정했습니다.
- 후보와 오버레이 슬롯의 연결 상태를 객체 참조뿐 아니라 후보 ID로도 검증하도록 보강했습니다.
- 슬롯 목록 전체 초기화 시 연결된 섹션과 오버레이 슬롯도 함께 제거되도록 수정했습니다.
- 프로필 자동 저장이 불필요하게 로그를 반복 생성하거나 변경 없는 종료 시 파일을 덮어쓰지 않도록 보완했습니다.

## 참고 사항

- 이 버전은 beta 버전이며, 사용 환경에 따라 추가 문제가 발생할 수 있습니다.
- WGC 사용 시 Windows 동작에 따라 노란색 캡쳐 테두리가 표시될 수 있습니다.
